QuickRead

The project involves developing a Chrome extension that helps users better interact with web content by providing key functionalities:
Content Summarization: The extension will use API-based services to extract and summarize the key points from web content. The user can trigger the summary either through a button or a context menu option.

Line following Robo

Objective: A line-following robot autonomously tracks a line on the ground.
Components: It uses a microcontroller, sensors (e.g., infrared), and motors with a motor driver.
Functionality: Sensors detect the line, and the robot adjusts its movement to stay on track.
Control System: The microcontroller processes sensor data and controls the motors to correct the robot’s path.
